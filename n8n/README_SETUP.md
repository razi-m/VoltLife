# VoltLife — n8n Logistics Workflow Setup Guide

This guide walks you through importing the post-payment logistics simulation workflow into your local n8n instance and connecting it to the VoltLife backend.

---

## 🛠️ Step 1: Install and Run n8n
If you do not have n8n running locally, you can start it using one of the following methods:

### Option A: Running via npm (npx)
Execute the following command in your terminal:
```bash
npx n8n
```

### Option B: Running via Docker
Execute the following command:
```bash
docker run -it --rm -p 5678:5678 n8nio/n8n
```

Once started, open [http://localhost:5678](http://localhost:5678) in your browser and complete the initial owner registration.

---

## 📥 Step 2: Import the Workflow
1. Navigate to the **Workflows** section in n8n.
2. Click on the **options menu** (three dots in the top-right corner) and select **Import from File**.
3. Choose the [voltlife_logistics_workflow.json](file:///c:/Users/Razi/Claude/Projects/VoltLife/n8n/voltlife_logistics_workflow.json) file located in this directory.
4. The workflow canvas will render a Webhook trigger, a payload builder function, and a series of HTTP Request callback nodes separated by Wait nodes.

---

## 🔗 Step 3: Configure Webhook and Environment
1. Click on the **Webhook** node at the beginning of the workflow.
2. Copy the **Production URL** or **Test URL** generated by n8n.
   - It will look like: `http://localhost:5678/webhook/voltlife/order-paid`
3. Open the `backend/.env` file in the VoltLife project and configure the parameters:
   ```env
   N8N_ENABLED=true
   N8N_WEBHOOK_URL=http://localhost:5678/webhook/voltlife/order-paid
   ```
4. Restart your VoltLife backend service to apply the environment changes.
5. In n8n, click **Active** (toggle in the top-right corner) to publish/activate the workflow.

---

## 🧪 Step 4: Run a Demo Purchase
1. Authenticate as a buyer and request a quote.
2. Complete checkout via `/api/v1/payments/mock-confirm`.
3. The backend will detect `N8N_ENABLED=true` and POST the order to the n8n webhook.
4. Watch the live execution path in n8n as the workflow advances the order status dynamically every 5 seconds, logging tracking events on the VoltLife backend and broadcasting progress over WebSockets!
